# EXTRA CREDIT - Random and Probabilistic L-Systems:
For the extra credit, I randomized the growth of the tree L-system shown in Figure 1.24, page 25 of the book  "The Algorithmic Beauty of Plants". Because the tree grows up, I moved the turtle to the bottom of the window when capturing this system. 

To Randomize angles and distances, I introduced new variables as follows: 
  ## f : move forward a distance randomized within a 30% range (randomized distance is at least 30% and at most 130% the specified distance . 
  ## b: move backward a random distance within a 30% range
  ## r: turn right a random angle within a 20% range.
  ## l: turn left a random angle within a 20% range. 
Then, I replaced F, +, and - in the system with f, l,and r respectivly. I also added the constants "/" and "*" to divides and multiplies the move distance by 2, which I used to change and track the distance betweeen pushes/pops. This fixed a branch sizing issue I had as the vocab X was not clearly defined in the book so I made it the same as F, but with a different specified distance.
For simplicity and to keep this short, I applied both the distance and angle randomization at the same time. This made the resulting tree, shown below, less symmetrical than the original, but still not realistic enough. 


Next, I implemented probablistic Lsystem which takes an array of strings instead of a string in its rules hashmap, and randomly selects a string from the array at each instance of the rule application. I chose to pass different subsegments from the original rule as possible replacements, which resulted in a stochastic bush-like tree. I manualy applied the rule to some X's in the axiom to start with an axiom large enough to generate a bush.
